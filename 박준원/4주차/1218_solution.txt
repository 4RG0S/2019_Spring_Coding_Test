2. 문자열의 길이
3. 문자열

입력 : 문자열의 길이, 문자열
출력 : 1 / 0 (유효 여부)

--------------------------------------------------------------------

1. 문자열을 담을 스택을 생성한다.

2. 문자열을 하나씩 읽는다.
	2-1. 열린 괄호이면 스택에 추가
	2-2. 닫힌 괄호이면 스택의 마지막 괄호가 매칭이 되는지 확인
		2-2-1. 매칭이 되면 스택의 마지막 괄호를 삭제
		2-2-2. 매칭이 되지 않으면 반복을 종료하고 0을 반환

4. 문자열을 다 읽었을 때 스택이 null이라면 1을 반환 아니면 0을 반환
